Downloads

This site is used by Gauge to get download metadata and check updates for plugins, templates and Gauge executable.

These URLs are used within Gauge for downloading updates

https://downloads.gauge.org/plugin

Used by Gauge to get the metadata of plugins. https://downloads.gauge.org/plugin/{plugin_name} gives the metadata for the plugin.

https://downloads.gauge.org/stable

It returns a shell script which can be used to install Gauge using curl or wget.
